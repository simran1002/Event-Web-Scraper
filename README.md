## Setup

1. **Install Node.js:** [https://nodejs.org/](https://nodejs.org/)

2. **Clone the repository:**

    ```bash
    git clone https://github.com/yourusername/Event-Web-Scraper.git
    cd Event-Web-Scraper
    ```

3. **Install dependencies:**

    ```bash
    npm install
    ```
